# Apple iTunes Music Analysis

## Project Overview
This project analyzes the Apple iTunes music store dataset to understand customer behavior, music preferences, and sales performance.

The goal of this project is to transform raw transactional data into meaningful insights using SQL and data visualization tools.

The analysis focuses on identifying popular music genres, top customers, revenue trends, and employee sales performance.

---

## Problem Statement
Apple iTunes manages a large digital music store with thousands of tracks, artists, and customers worldwide.

As the business grows, it becomes important to analyze the data to understand purchasing behavior, music popularity, and overall sales performance.

This project performs data analysis on the iTunes relational database to generate actionable business insights.

---

## Project Objectives
- Analyze customer purchasing behavior
- Identify the most popular music genres and artists
- Evaluate sales performance across different countries
- Track revenue trends over time
- Build a dashboard to visualize key business metrics

---

## Dataset Description
The project uses the Apple iTunes relational dataset containing multiple tables such as:

- Customers
- Employees
- Invoices
- Invoice Lines
- Tracks
- Albums
- Artists
- Genres
- Media Types
- Playlists

These tables are connected using primary keys and foreign keys to form a relational database.

---

## SQL Analysis
SQL queries were used to perform exploratory and advanced data analysis including:

- Total revenue calculation
- Customer distribution by country
- Top spending customers
- Most popular music genres
- Top selling artists
- Employee sales performance
- Monthly and yearly revenue trends

Advanced SQL techniques such as CTEs, Window Functions, and Ranking were used to generate deeper insights.

---

## Dashboard
An interactive dashboard was created to visualize key business metrics such as:

- Total Revenue
- Total Orders
- Total Customers
- Monthly Revenue Trend
- Top Customers
- Most Purchased Genres
