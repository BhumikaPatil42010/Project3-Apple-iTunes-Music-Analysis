# Project Report

## Overview
This folder contains the final project report for the **Apple iTunes Music Analysis** project.

The report explains the complete workflow of the project including database design, SQL analysis, dashboard creation, and business insights derived from the dataset.

---

## Report Contents

The report includes the following sections:

- Project Overview  
- Problem Statement  
- Business Goals  
- Database Design and ER Diagram  
- Data Import Process  
- SQL Exploratory Data Analysis  
- Advanced SQL Analytics  
- Power BI Dashboard Creation  
- Business Insights and Recommendations  
- Conclusion  

---

## Purpose of the Report

The purpose of this report is to document the complete data analysis process and present the key findings in a structured format.

It demonstrates how SQL and Power BI were used to transform raw data into meaningful business insights.
