# Dashboard (Power BI)

## Overview
This folder contains the business intelligence dashboard created using Microsoft Power BI. The dashboard visualizes key insights derived from the Apple iTunes Music Analysis dataset.

The purpose of this dashboard is to provide a clear view of sales performance, customer activity, and music trends in the iTunes store.

---

## Dashboard Features

### Total Revenue
Displays the total revenue generated from music purchases.

### Total Orders
Shows the total number of invoices generated.

### Total Customers
Displays the number of customers in the dataset.

### Monthly Revenue Trend
A line chart that shows how revenue changes over time.

### Top Customers
A bar chart that identifies the highest spending customers.

### Most Purchased Genres
Displays which music genres are most popular among customers.
