# SQL Queries

## Overview
This folder contains all the SQL queries used in the **Apple iTunes Music Analysis** project.

The queries were written to create the database schema, validate the dataset, perform exploratory data analysis (EDA), and generate advanced analytical insights from the iTunes music store data.

The analysis was performed using MySQL.

---

## SQL Workflow

The SQL analysis in this project was divided into multiple steps:

### 1. Database Setup
Creation of database and tables using SQL.

Tables created include:

- media_type
- genre
- artist
- employee
- customer
- album
- track
- playlist
- playlist_track
- invoice
- invoice_line

Primary keys and foreign keys were used to establish relationships between tables.

---

### 2. Data Validation
Queries were executed to verify that the data was imported correctly.

Examples:
- Record count checks
- Relationship validation between tables
- Checking for missing values
- Identifying inactive customers or unused tracks

---

### 3. Exploratory Data Analysis (EDA)
SQL queries were used to understand the dataset and extract meaningful insights such as:

- Total revenue generated
- Total number of customers
- Total tracks available
- Customer distribution by country
- Revenue by country
- Top spending customers

---

### 4. Music & Product Analysis
Analysis related to music content in the store.

Examples include:

- Most popular music genres
- Top selling artists
- Most purchased tracks
- Playlist popularity

---

### 5. Employee Performance Analysis
Queries were written to evaluate employee sales performance by calculating revenue generated by customers managed by each employee.

---

### 6. Revenue Trend Analysis
Revenue trends were analyzed using SQL queries.

Examples:

- Monthly revenue trend
- Yearly revenue trend
- Revenue by media type

---

### 7. Advanced SQL Analytics
Advanced SQL techniques were used to generate deeper insights.

These include:

- Common Table Expressions (CTE)
- Window Functions
- Ranking (RANK, DENSE_RANK)
- Customer segmentation
- Revenue growth analysis

These techniques helped identify top customers, best performing artists, and revenue patterns.
